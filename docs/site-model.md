# Protheus Labs Site Model

Reviewed against live public references on April 10, 2026.

## Reference profile

- Anthropic: calm editorial spacing, trust-heavy structure, restrained hierarchy, premium serif/sans contrast.
- Ollama: minimal install-first storytelling, bright light surfaces, low-friction product page rhythm.
- Hugging Face: product-forward navigation, modular showcase sections, dense but readable information cards.
- OpenClaw: stronger product personality, integration storytelling, dark-mode confidence, more vivid section transitions.
- OpenFang: utility-first clarity for docs-like surfaces and straightforward technical framing.

## Refined model for Protheus

- Tone: legitimate, product-led, local-first, operator-grade.
- Palette: off-white and graphite neutrals with a muted steel-blue accent, not flashy and not startup-purple.
- Typography: Manrope for body/UI, Fraunces for high-trust display moments.
- Layout: sticky company header, immediate product access, strong hero, card-based product grid, clear dedicated product page.
- Motion: subtle fades, soft hover lift, no novelty animations.
- Product page rule: move from value proposition to install flow within the first screen.
- Footer rule: social icons only render when links are actually available.

## Implementation decisions

- Dedicated `/infring` page because InfRing already exposes a credible public quick-start and install profile model.
- `Products` dropdown in the top bar to support a multi-product company posture from day one.
- System-sensed light/dark presentation with an Ollama-like light mode baseline and a simple graphite dark mode.
- Placeholder circular brand mark until a real Protheus Labs identity asset is ready.
